## 运行

这个工作区使用 python3.10 + venv 构建
虽然python版本错误可能会导致无法运行代码
但是，大多数时候都是兼容的，无需担心这个问题

*如果没有 venv 文件夹，就先运行 InitVenv.bat* 
<u>Mac的初始化工具暂时不做</u>

想要启动这个工作区的代码
可以运行 StartLstmTest.bat
这个批处理文件会激活虚拟环境并运行 LstmNetTest.py
其他Start\*.bat文件也是类似的

你也可以手动激活虚拟环境然后运行代码
虚拟环境路径: .\venv\Scripts\activate

在pycharm,vscode等IDE中
可以直接指定python解释器为虚拟环境中的解释器
路径: .\venv\Scripts\python.exe
然后直接运行代码文件即可
